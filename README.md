## What I'm building

### 🎯 [Product Org Maturity — ACE Framework](https://github.com/jenakap/product-org-maturity)
An interactive assessment tool that helps product organizations understand where they are, where they need to be, and what's standing in the way.

Built around the **ACE Framework** — three pillars that cut through the noise:
- **A — Alignment**: Are we building the right things for the business?
- **C — Capability**: Do we have what it takes to sustain it?
- **E — Execution**: Can we deliver reliably and fast?

Rate your org across 12 dimensions, benchmark against industry standards, and get AI-powered recommendations tailored to your company stage and constraints. The kind of tool I wish existed in every CPO onboarding conversation I've ever had.

> React + TypeScript + Vite · Recharts · shadcn/ui · Tailwind CSS

---

### 🗺️ [AI Roadmap Builder](https://github.com/jenakap/portfolio-w-git)
A roadmap tool that actually helps you decide what to build — not just track it. Draft initiatives, and let AI (Claude) score them, suggest priorities, and surface strategic tradeoffs you might be missing.

Includes scenario planning so you can compare "what if we sequenced it this way?" side-by-side, plus visualizations (radar, bubble, pie charts) that make the tradeoffs obvious in a room full of stakeholders. Exports included for when it needs to leave the browser.

Built because roadmap debates are usually about gut feeling vs. gut feeling. This puts AI in the room as a neutral analyst.

> React + TypeScript · Claude API · Tailwind CSS · Recharts

---

### 📁 [Portfolio Lifecycle Manager](https://github.com/jenakap/portfolio-manager)
A web app for product leaders who manage multiple products and need to see the full picture — not just what shipped, but what's healthy, what's at risk, and where to move resources.

Tracks products through five lifecycle stages (Incubate → Early Adoption → Expand → Perform → Evolve) with real-time metrics (ARR, growth, R&D ratios, win rates), a reallocation calculator to simulate resource shifts, and an AI-powered chat interface that gives strategic recommendations based on your actual portfolio data.

Built because most portfolio reviews happen in spreadsheets and end in politics. This puts the data and the analysis in one place.

> React + TypeScript · Recharts · Claude API · Upstash Redis · Tailwind CSS
